HTML - podstawowe informacje

Podstawowe znaczniki HTML'owe

  • <html>
  • <head>
  • <body>
  • <main> - główna część dokumentu
  • <section> - sekcje dokumentu
  • <article> - samodzielne treści
  • <header> - nagłówek dokumentu / sekcji / artykułu
  • <footer> - stopka dokumentu / sekcji
  • <h1-h6> - tekstowe nagłówki tytułowe - dokumentu / sekcji
  • <p> - paragraf / akapit
  • <br> - nowa linia
  • <b> - pogrubienie tekstu (brak znaczenia semantycznego)
  • <strong> - podkreśla wysoką istotność i poważność frazy w nim zawartej
  • <i> - kursywa (brak znaczenia semantycznego)
  • <em> - podkreśla, że tekst może mieć inne znaczenie (np. ironia)
  • <a> - linki / łącza wewnątrz dokumentu
  • <nav> - nawigacja
  • <img> - obrazek o podanym adresie

Budowa znaczników HTML'owych

  • <znacznik atrybut=”wartość”>treść</znacznik>
  • <a href="" target="">opis łącza</a> a - linki / łącza wewnątrz dokumentu
  • <element id="moj-identyfikator"> - atrybut id - identyfikuje element (przydaje się do nawigacji w obrębie dokumentu)
  • <img src="" alt=""> - obrazek o podanym adresie (src - ścieżka do obrazka; alt - opis treści obrazka)

CSS - czyli mój styl

CSS - podstawowe informacje

CSS = Cascading Style Sheets - Kaskadowe arkusze stylów
CSS Odpowiadają za prezentację treści strony.CSS wpływają na wygląd elementów HTML na ekranie, wydrukach itp

Jak stylować

Sposoby zapisu styli:

Stylowanie inline - <znacznik style=”color:yellow;”>treść</znacznik>
Stylowanie z użyciem:
  • #id - identyfikator elementu
  • .class - klasa
  • element - element np.: body, header

Style umieszczone w sekcji <style> w pliku '*.html' (nie zalecane, nie da się tego używąc na tronach mających więcej niż HELLO WORLD :)) lub w osobnym pliku np.: *.css.
Definuijemy w sekcji <head>:
<link href="style.css" rel="stylesheet" type="text/css" />

Gramatyka pliku ze stylami

# (hash) - selektor identyfikatora elemantu - id
#name {
 właściwość: wartość;
}

. (kropka) - selektor klasy - class
.name {
 właściwość: wartość;
}

(brak "przedrostka") - selektor elementu
name {
 właściwość: wartość;
}
